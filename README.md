HIDmon Kai
===============

HIDmon Kai is a modified version of HIDmon.

This repository has these module.
1. Firmware for ATTiny2313.
2. Ruby module for control HIDmon Kai which is worked on Win32, Linux and mac.
3. AVR Writter program.

Goal
---------------

HIDmon Kai is
* USB-GPIO
* AVR Writter
